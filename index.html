Bem-vindo ao meu projeto!

Este site reúne todas as atividades desenvolvidas nas aulas de Desenvolvimento de Interfaces. A proposta é praticar conceitos essenciais como menus, listas, tabelas, formulários, imagens e vídeos, além de aplicar HTML semântico, utilização de IDs e classes, criação de cards e até mesmo a publicação na web por meio de plataformas como o Netlify.

No menu superior você pode acessar as páginas e visualizar cada atividade de forma individual. Esta página inicial serve como um guia geral e também como porta de entrada para o conteúdo completo.

HTML Semântico

Uso correto de tags como <header>, <section>, <article> para estruturar melhor o conteúdo.

Estilização com CSS

Criação de menus, cores harmonizadas em rosa, efeitos de hover e layout responsivo.

Publicação

Aprendizado sobre como hospedar o site usando o Netlify e torná-lo acessível online.
